Speed and Velocity

In the previous section on Position, Distance, and Displacement, we explored how coordinate systems are used to describe the position of an object in space at a given time, and were introduced to the position function. Now, we would like to go beyond that and begin describing the behaviour of that object as it moves through space.

Speed

Naturally, as an object moves through space, we would like to know the distance that the object moves over a period of time. In other words, we would like to know the rate at which its position changes over time. This is known as the speed of the object.

In order to calculate the speed of an object, we take the total distance travelled by that object and divide it by the time that it took for the object to travel that distance:

$$ \frac{ \Delta p }{ \Delta t } $$

The resulting value tells us exactly how far an object moves per unit of time. For example, an object that is moving at a speed of

$$ 10 \ \frac{m}{s} $$

will move 10 metres every second.

The value of speed will always be a positive number.
