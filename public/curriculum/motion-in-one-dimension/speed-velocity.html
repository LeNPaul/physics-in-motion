Speed and Velocity

In the previous section on Position, Distance, and Displacement, we explored how coordinate systems are used to describe the position of an object in space at a given time, and were introduced to the position function. Now, we would like to go beyond that and begin describing the behaviour of that object as it moves through space.

Speed

Naturally, as an object moves through space, we would like to know the distance that the object moves over a period of time. In other words, we would like to know the rate at which its position changes over time. This is known as the speed of the object.

In order to calculate the speed of an object, we take the total distance travelled by that object and divide it by the time that it took for the object to travel that distance:

$$ speed = \frac{p_2 - p_1}{t_2 - t_1} = \frac{ \Delta p }{ \Delta t } $$

The resulting value tells us exactly how far an object moves per unit of time. For example, an object that is moving at a speed of

$$ 10 \ \frac{m}{s} $$

will move 10 metres every second.

The value of speed will always be a positive number.

Average and Constant Speed

Notice above that we did not capture any information regarding the speed of the object at any specific point in time. As the object moves through space, it is possible for the speed of the object to be faster or slower at certain times within the period of time that the object is moving from the starting position to the final position. It is also possible for the speed of the object to be constant throughout that period of time. What we had captured above was the average speed of the object.

When we talk about average speed, we only know the total distance travelled, and the time that it took to travel that distance. An object may have constant speed throughout that time interval, or variable speed, and it is possible for two different objects to have the same average speed, but different speeds throughout the periods of time that they were travelling.

Instantaneous Speed

In order to capture the full picture of how an object moves from its starting position to its final position within a period of time, we consider what we call instantaneous velocity, which is a snapshot of what an object's speed is at any specific point in time.

In the case of constant speed, the object's instantaneous speed will be the same at every point in time. In the case of variable speed, the object's instantaneous speed may be different at different points in time. If both of the above objects travel the same distance at the time same, their average speed will be the same, despite having different instantaneous speeds throughout the periods of time that they were travelling.

Through instantaneous speeds, we are able to capture a more complete picture of an object's displacement in time.

Average and Constant Velocity

When we refer only to the speed of an object, we do not include any information on the direction in which the object is travelling. When we know both the speed and direction of an object, we call this value the velocity of the object.

Unlike speed, values of velocity can be either positive or negative. In one dimension, this tells us the direction that the object is travelling in, as there are only two directions in which the object may move in. We consider positive velocities one direction, and negative velocities the opposite direction. The magnitude of velocity is equal to speed.

Average velocity is the total dispacement of an object divided by the interval of time that it took for the object to be displaced:

$$ v_{average} = \frac{p_2 - p_1}{t_2 - t_1} = \frac{\Delta p}{\Delta t} $$

Similar to average speed, it is possible for the velocity of an object to be either constant, or variable through the time interval that is used to calculate its average velocity.

Instantaneous Velocity

Similar to instantaneous speed, instantaneous velocity is a snapshot of what an object's velocity is at a specific point in time.

More formally, the velocity of an object is defined as the derivative of its position function:

$$ v(t) = \lim_{\Delta t \to 0} \frac{\Delta p}{\Delta t} = \frac{d p(t)}{dt} $$

This describes how fast something moves, and in which direction it is moving at any point in time. On a graph, the instantaneous velocity of an object is equal to the tangent of the position function at a point in time.
