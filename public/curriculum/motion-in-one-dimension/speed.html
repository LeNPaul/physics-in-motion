Speed

In the previous section on Position, we had explored one of the tools that we use to describe the position of an object in spce at a given time, and were introduced to the position function. Now, we would like to go beyond on that start describing the behaviour of that object as it moves through space.

Naturally, as an object moves through space, we would like to know the rate at which its position changes over time. In other words, we would like to know the speed of the object.

In order to calculate the speed of an object, we take the total displacement of the object, and divide it by time that it took for the object to make that displacement.

$$ \frac{ \Delta p }{ \Delta t } $$
